如何使用Github的301重定向

发表于 2018-05-04 更新于 2020-10-01 分类于 Github ， 网站

301重定向

Github为大家提供了301重定向的功能，这是我很喜欢Github的原因之一。

打个比方，我以前在服务器搭建的博客，如果把ly554.com和www.ly554.com都指向到同一个服务器，但搜索引擎会认为这是两个不同的网站，从而导致权重分散。

自从把博客托管到Github后，忽然发现Github可以对域名进行301重定向。

在Github只需要在根目录创建一个名为CNAME的文件，填入你主要使用的域名，我写的是www.ly554.com。

然后在域名管理中设置记录类型为CNAME，将ly554.com和www.ly554.com都指向到xxxx.github.io。这样就完成了重定向。(xxxx改为你的Github用户名)
